This is a chrome extension that will autofill forms on a webpage.
This is a simple , lightweight , fast and free autofill extension that will help you to fill forms on a webpage. 

How to use:-

1. Click on the extension icon
2. Click on "Save Current Form"
3. Enter a name for the form
4. Click on "Save Form"
5. Click on "Fill" to autofill the form
